# Ship checklist

Use before handoff or merge to `main`.

## Accessibility

- [ ] Landmarks: `header`, `main`, `footer` sensible; `nav` has accessible name when multiple exist.
- [ ] Heading order is logical; interactive controls keyboard reachable.
- [ ] Images have appropriate `alt` text.

## Performance

- [ ] No enormous unoptimized hero assets in `public/` or imports.
- [ ] Routes that are very heavy considered for lazy loading.

## SEO / metadata

- [ ] Page titles and meta descriptions updated for key routes.

## Content / brand

- [ ] Copy matches approved source; logos and marks used per `revamp-input/brief.md`.
